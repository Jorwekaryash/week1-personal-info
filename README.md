# Personal Information Manager

## Project Description

This is a beginner Python program that stores static personal information and collects user input to display it in a clean, formatted output

## What I Learned

- Using variables to store different data types
- Taking user input and validatong it
- Formatting output using f-strings
- Writing simple, readable Python program

## How to Run This Program

1. Make sure Python is installed on your system
2. Open a terminal or command prompt
3. Navigate to the project folder
4. Run the command: python personal_info.py
5. Follow the on-screen instructions

## Features

- Stores static personal information
- Collects user input with basic validation
- Displays formatted output with clear sections
- Calculates age in months
